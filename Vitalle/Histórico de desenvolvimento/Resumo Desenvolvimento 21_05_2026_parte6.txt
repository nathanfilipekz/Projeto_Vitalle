3. AJUSTE NO LIMITE DE CADASTRO DE USUÁRIOS
============================================================
Problema:
  A plataforma bloqueava qualquer novo cadastro pois já existia
  1 usuário no banco (Mayke Ribeiro Barbosa, criado em 20/05/2026).
  A função hasAnyUser() retornava true para count > 0.

Solução:
  Limite alterado de 1 para 2 usuários por instância MVP.

Arquivo: frontend/src/services/auth-service.ts
  Antes:  return (count ?? 0) > 0;
  Depois: return (count ?? 0) >= 2;

Arquivo: frontend/src/app/(auth)/register/page.tsx
  Mensagem atualizada:
  Antes:  "Já existe um usuário cadastrado nesta ferramenta.
           Apenas um usuário pode ser criado por instância."
  Depois: "O limite de 2 usuários por instância foi atingido."
